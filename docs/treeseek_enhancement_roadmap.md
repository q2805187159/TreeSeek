# TreeSeek Enhancement Roadmap

## Purpose

This document tracks the staged evolution of TreeSeek from a local structured retrieval toolkit into a more complete production-ready long-document retrieval system.

## Current Focus

The current codebase already supports:

- PDF tree extraction
- Markdown tree extraction
- local query index generation
- query-only retrieval
- BM25-lite ranking
- snippet/highlight output
- explain mode

## Next Recommended Directions

### Phase 3

- Multi-document corpus indexing
- Interactive CLI mode
- HTTP API / service wrapper
- Metadata filtering
- Domain-specific presets

### Phase 4

- Better PDF layout understanding
- Table-aware splitting
- OCR-assisted extraction for scanned PDFs
- richer evidence extraction

## Core Scenario Targets

- financial reports
- legal and compliance documents
- enterprise knowledge bases
- technical manuals
- operations and maintenance documentation
